# kurento-player-rtsp2webrtc-jsclient
 基于KurentoMediaServer，tutorial-java里面的kurento-player改造成纯js客户端，用于在网页中播放rtsp视频。

使用：

1.安装并启动KMS。

2.使用IDEA启动服务端。

3.打开前端页面（可以不同放到IIS中），并连接服务端，播放视频。
